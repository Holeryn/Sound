# Sound
# Run this software into a Dos environment
a program that generates the imperial march through the motherboard buzzer

# Compile
nasm -f obj sound.asm

tlink sound.obj + exelib.obj

# Or
Use the make.bat:

$ Make

Or

$ Make.bat
